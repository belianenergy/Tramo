## ✅ Candidatos para UI Amigable, Moderna e Avanzada

### 1. Ambrook ⭐⭐ MELHOR CANDIDATO

- **URL:** https://ambrook.com
- **ID Refero:** `f3c955e4-0fea-462d-b05f-868552e9628c`
- **Esquema:** Light
- **Industria:** Fintech (xestión financeira agrícola)
- **Paleta:**
  - Fondo: Crema cálido `#fcfaf1` (non branco frío)
  - Texto: Marrón escuro `#211b15` (non negro frío)
  - Secundario: Verde terroso `#434f40` (natureza, enerxía)
  - Accent: `#efe9e0` (beige suave)

- **Por que é o mellor para Tramo:**
  - ✅ **Amigable**: tons cálidos, non fríos como o negro/branco puro
  - ✅ **Moderno**: tipografía limpa, espazos xenerosos, cards sutís

  - ✅ **Enerxía**: a paleta verde-terra-crema conecta con sostenibilidade/enerxía
  - ✅ **Profesional pero humano**: fintech agrícola = seriedade + calidez
  - ✅ **Non parece IA**: é un produto real con identidade propia
  - ✅ **B2B**: dirixido a profesionais, non a consumers

## 🏆 Recomendación: Ambrook como Referencia Principal

Ambrook é o mellor match para Tramo porque:

1. **Non é dark** → light, cálido, amigable
2. **É avanzado** → produto financeiro con datos reais, táboas, dashboard
3. **É moderno** → tipografía limpa, espazos ben medidos, cards sutís
4. **Conecta con enerxía** → paleta verde-terra-crema = sostenibilidade
5. **É B2B profesional** → dirixido a xestores/profesionais
6. **Ten personalidade propia** → non parece xenérico nin IA
7. **Está en Refero** con DESIGN.md extraíble

### Nova Composición Proposta

| Sección Tramo | Referencia Refero | Por que |
|---|---|---|
| **Hero + Landing xeral** | **Ambrook** | Crema cálido, profesional, amigable |
| **Features / Produto** | Notion | Limpo, organizado, moderno |
| **Dashboard / App preview** | Ambrook + Superhuman | Datos reais con toque premium |
| **Pricing / Pilot** | Superhuman | Sensación de valor, exclusividade |
| **Contacto / CTA** | Loom | Amigable, accesible, cores vivas |
